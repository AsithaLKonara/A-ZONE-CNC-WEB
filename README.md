# 🚀 A ZONE CNC & Laser Solutions

Welcome to the official repository for **A ZONE CNC & Laser Solutions**, a high-precision fabrication service specializing in CNC cutting, laser engraving, and custom fabrication. This website is designed with a futuristic, cinematic 3D aesthetic to showcase our capabilities and high-tech craftsmanship.

## ✨ Project Vision

A ZONE aims to bridge the gap between traditional fabrication and futuristic design. Our digital presence reflects the precision of our machines and the creativity of our solutions.

## 🏗 Website Structure

- **🏠 Home**: A cinematic hero section with 3D backgrounds and brand identity.
- **🛠 Services**: Detailed overview of CNC Cutting, Laser Engraving, and Custom Fabrication.
- **🖼 Portfolio**: A high-end gallery showcasing real-world projects and craftsmanship.
- **ℹ️ About**: Our story, technology, and commitment to quality.
- **📞 Contact**: Streamlined quote request system and AI-powered assistant.

## 🛠 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **3D Engine**: [Three.js](https://threejs.org/) via [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **UI Components**: Radix UI / Shadcn UI (Planned)

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AsithaLKonara/A-ZONE-CNC-WEB.git
   cd A-ZONE-CNC-WEB
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
